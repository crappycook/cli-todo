# A Simple Command Line TODO List Tool Written In Rust

## requirements

```
rustc 1.66.0
cargo 1.66.0
sqlite3 3.39.4
```

## Usage

```
Usage: cli-todo <COMMAND>

Commands:
  add   Add todo item
  list  List all todo items
  rm    Remove todo item by id
  help  Print this message or the help of the given subcommand(s)
```
